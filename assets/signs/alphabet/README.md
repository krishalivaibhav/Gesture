# Sign Template Images

Place your static ASL sign reference images in this folder.

## Required naming

- Use uppercase letter filenames:
  - `A.png`, `B.png`, ... `Y.png`
- v1 excludes motion letters:
  - `J` and `Z` are intentionally not used.

## Accepted extensions

- `.png`
- `.jpg`
- `.jpeg`

## Tips for better template quality

- One clear hand only in the image.
- Good lighting and plain background.
- Hand fully visible (no cropped fingertips).
- Keep sign centered in frame.

The app extracts landmarks from these images at startup and uses template matching for Sign mode.
